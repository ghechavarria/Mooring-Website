# Primary silver button (`.btn-primary-silver`)

Defined in [`src/index.css`](../src/index.css) (`@layer components`). Used for **`Request a briefing`** ([`Header.tsx`](../src/components/Header.tsx), [`Hero.tsx`](../src/components/Hero.tsx), mobile nav) and **`Email our solutions team`** ([`ContactCTA.tsx`](../src/components/ContactCTA.tsx)).

**Silver-toned** fill (`theme` colors **`accent`** / **`accent-light`**), **`text-slate-950`**, lift shadow, **`ring-accent`**, hover to **`accent-light`**, active **`translateY(1px)`**. Add spacing/size with utilities, e.g. **`px-4 py-2.5 text-sm`**, **`px-8 py-3.5`**.

Accent aligns with the **metallic logo tile**; warm **`organ`** neutrals remain for body copy and marble surfaces.
