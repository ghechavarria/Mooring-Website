# Hero ECG (merged)

The standalone **`HeroEcgRibbon.tsx`** component was **removed**. The scrolling ECG waveform now lives inside **`HeroSessionPulse`** — **`variant="integrated"`** for the main hero slab, and the **`panel`** variant for the card layout.

See [hero-session-pulse.md](./hero-session-pulse.md) for behavior, reduced motion, and SVG details.
