# Hero section background (`Hero.tsx`)

The marketing hero sits on a **true ink–navy** field so it aligns with the **logo tile** (`#020617`, Tailwind `slate-950` family) instead of a flat neutral gray.

## Surface

- **Section:** `bg-[#020617]` with `border-slate-800/90` for a crisp cool edge. **`pt-0`** so the session strip meets the site header with no extra padding band; **`scroll-mt-24`** for in-page anchors below the sticky header.
- **Radial wash (decorative):** Layered **`sky`**, **`#1e70d6`**, **`slate-900`**, and faint **`accent`** — the top sky ellipse is anchored at **`50% 12%`** (not above the fold) so it does not read as a thin bright line along the top edge. Remaining ellipses match the prior cool wash.
