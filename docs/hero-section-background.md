# Hero section background (`Hero.tsx`)

The marketing hero sits on a **true ink–navy** field so it aligns with the **logo tile** (`#020617`, Tailwind `slate-950` family) instead of a flat neutral gray.

## Surface

- **Section:** `bg-[#020617]` with `border-slate-800/90` for a crisp cool edge.
- **Radial wash (decorative):** Layered **`sky`**, **`#1e70d6` (ERP blue)**, **`slate-900`**, and a faint **`accent`** mist so depth reads **blue-cool**, not brown or muddy.
