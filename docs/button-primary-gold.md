# Primary gold button (`.btn-primary-gold`)

Defined in **`src/index.css`** (`@layer components`). Used for **`Request a briefing`** ( **`Header`**, **`Hero`**, mobile nav) and **`Email our solutions team`** (**`ContactCTA`**).

Gold fill, **`text-slate-950`**, lift shadow, **`ring-gold`**, hover to **`glow`**, active **`translateY(1px)`**. Add spacing/size with utilities, e.g. **`px-4 py-2.5 text-sm`**, **`px-8 py-3.5`**.
